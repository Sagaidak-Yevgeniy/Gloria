# Gloria Voice Assistant: Учебный курс по созданию голосового ассистента

![Gloria Voice Assistant]([https://via.placeholder.com/800x200.png?text=Gloria+Voice+Assistant](https://img.freepik.com/premium-vector/digital-chat-bot-robot-assistant-customer-support-concept-virtual-conversation-assistant-getting-help-vector-illustration-isolated-white-background_714605-1045.jpg))

**Описание курса:**  
Этот курс предназначен для студентов и преподавателей, которые хотят изучить создание голосового ассистента на Python с использованием технологий распознавания и синтеза речи. Курс демонстрирует полный цикл разработки: от прослушивания команд через микрофон до воспроизведения речи, управления музыкой и отображения эмоций через GUI.  

---

## Что изучает студент
- Синтез речи с помощью **Silero TTS**  
- Распознавание речи с использованием **Vosk**  
- Создание интерактивного ассистента с GUI  
- Управление музыкой и медиа-командами через голос  
- Основы многопоточности в Python  
- Добавление новых команд и расширение функционала  

---

## Структура курса

### Урок 1: Синтез речи (Silero TTS)
- Создание функции `say()` для генерации речи  
- Проигрывание аудио через `pygame`  
- Разделение функций генерации и воспроизведения аудио  

### Урок 2: Распознавание речи (Vosk)
- Подключение микрофона через `pyaudio`  
- Получение текста из речи пользователя с помощью `KaldiRecognizer`  
- Обработка команд ассистента на основе распознанного текста  

### Урок 3: Полный голосовой ассистент Gloria
- Объединение TTS и распознавания речи  
- Интерактивный GUI с эмоциями  
- Управление музыкой и звуком  
- Возможность задавать вопросы ассистенту  
- Добавление новых команд в словарь `cm.commands`  

---

## Примеры команд для ассистента
| Категория | Команда | Ответ ассистента |
|-----------|---------|-----------------|
| Приветствие | "Привет", "Здравствуйте" | Приветствие и запуск ассистента |
| Эмоции | "Эмоции" | Ассистент показывает все эмоции через GUI |
| Время | "Сколько времени?" | Ассистент сообщает текущее время |
| Музыка | "Включи музыку", "Дальше", "Пауза" | Управление воспроизведением музыки |
| Сказки | "Три поросенка", "Кукушка" | Ассистент проигрывает аудиосказку |
| Вопрос | "Как дела?", "Скажи что-нибудь" | Ассистент отвечает голосом с использованием TTS |
